Aviso Importante

Esta pagina web NO es oficial y solo es una pagina web de ejemplo creada con fines educativos y de demostracion. No tiene ninguna afiliacion con Michael Jackson, su patrimonio, sus herederos o cualquier entidad relacionada con el artista.

Todas las imagenes, videos y contenido multimedia utilizados en este sitio pertenecen a sus respectivos propietarios y se utilizan unicamente con fines ilustrativos.

1. Informacion Recopilada

Este sitio web de ejemplo no recopila ni almacena informacion personal de los visitantes. Cualquier dato almacenado en el navegador (como preferencias de idioma o tema) es puramente local y no se transmite a ningun servidor.

2. Cookies

Este sitio utiliza almacenamiento local del navegador (localStorage) para recordar las preferencias del usuario, como el idioma seleccionado y el modo oscuro/claro. Estas preferencias no se comparten con terceros.

3. Contenido de Terceros

Este sitio incrusta contenido de plataformas externas como Spotify, YouTube y Vimeo. Estos servicios pueden tener sus propias politicas de privacidad y pueden recopilar datos segun sus propios terminos.

4. Contacto

Este es un sitio de demostracion. Los formularios de contacto y el chat de Tawk.to son funcionales pero deben considerarse parte de una demostracion tecnica.
